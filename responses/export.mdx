Export session data for reporting, analysis, or integration with other tools.

## How to Export

<Steps>
  <Step title="Open Responses tab">
    Navigate to your scenario's **Responses** tab.
  </Step>
  <Step title="Apply filters (optional)">
    Use search, score range, or date filters to select which results to export.
  </Step>
  <Step title="Click Export">
    Click the **Export** button. An Excel file downloads immediately.
  </Step>
</Steps>

## Export Contents

The Excel file includes:

| Column | Description |
|--------|-------------|
| Participant Name | Who took the session |
| Overall Score | Average % across goals |
| Individual Goal Scores | One column per goal |
| Date | Session completion timestamp |
| Duration | Session length |

## Use Cases

- **HR reporting**: Share interview scores with hiring managers
- **Training tracking**: Monitor team progress over time in a spreadsheet
- **Client reporting**: Provide assessment results to enterprise clients
- **Data analysis**: Import into BI tools for deeper insights
