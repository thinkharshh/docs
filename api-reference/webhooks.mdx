# Webhooks

<Warning>
  **Coming soon.** You can configure a webhook URL today, but outgoing webhook delivery is not yet active. Use [polling](/api-reference/session-lifecycle) or [email notifications](/capabilities/notifications) to get session results in the meantime.
</Warning>

## Setup

Configure a webhook URL for your scenario via the **Settings** tab or the session options API:

```bash
curl -X PUT https://waterr.ai/backend/scenarios/{scenarioId}/session-options \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "webhook_url": "https://your-app.com/webhooks/waterr"
  }'
```

The `webhook_url` field is stored on the scenario's session options. When webhook delivery is enabled, we will POST to this URL when sessions complete.
