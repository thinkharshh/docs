## Get All Recordings

Retrieve all recordings for the authenticated user.

```bash
curl -X GET https://waterr.ai/backend/recordings \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

```json
[
  {
    "id": "recording-uuid",
    "meeting_id": "meeting-uuid",
    "url": "https://storage.example.com/recordings/abc123.mp4",
    "duration": 1800,
    "status": "completed",
    "created_at": "2024-01-15T11:10:00Z"
  }
]
```

## Get Recording by ID

<ParamField path="id" type="string" required>
  The recording UUID
</ParamField>

```bash
curl -X GET https://waterr.ai/backend/recordings/{id} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

## Get Recordings by Meeting ID

<ParamField path="meetingId" type="string" required>
  The meeting UUID
</ParamField>

```bash
curl -X GET https://waterr.ai/backend/recordings/meeting/{meetingId} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

## Get Recording URL

Get the playback URL (CloudFront CDN) for a recording. This is the recommended endpoint for embedding or linking to recordings.

<ParamField path="meetingId" type="string" required>
  The Daily meeting ID (returned as `daily_meeting_id` when creating a meeting)
</ParamField>

```bash
curl -X GET https://waterr.ai/backend/recordings/url/{meetingId} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

## Get Recording URL with Thumbnail

Same as above but also returns a thumbnail image URL for preview.

```bash
curl -X GET https://waterr.ai/backend/recordings/url-with-thumbnail/{meetingId} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

## Get Recordings by Scenario

Retrieve all recordings across meetings for a specific scenario.

<ParamField path="scenarioId" type="string" required>
  The scenario UUID
</ParamField>

```bash
curl -X GET https://waterr.ai/backend/recordings/scenario/{scenarioId} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```
