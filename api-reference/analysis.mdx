## Overview

Analysis provides detailed insights about a meeting, including performance metrics, strengths, growth areas, and goal-based evaluations.

<Note>
  **Analysis Timing:** Analysis is generated after a meeting ends. If you don't see analysis immediately, wait a bit and retry.
</Note>

## Get Analysis by Meeting ID

<ParamField path="meetingId" type="string" required>
  The meeting UUID
</ParamField>

```bash
curl -X GET https://api.waterr.ai/v1/analyses/meeting/{meetingId} \
  -H "Authorization: Bearer wai_<your_key>"
```

## Get All Analyses

Retrieve all analyses for the authenticated user.

```bash
curl -X GET https://api.waterr.ai/v1/analyses \
  -H "Authorization: Bearer wai_<your_key>"
```

## Get Analyses by Scenario

<ParamField path="scenarioId" type="string" required>

## Trigger Manual Analysis

Re-run analysis for a meeting that has already ended. Useful if the original analysis failed or you've updated the scenario's goals.

<ParamField path="meetingId" type="string" required>
  The meeting UUID
</ParamField>

```bash
curl -X POST https://api.waterr.ai/v1/analyses/trigger/{meetingId} \
  -H "Authorization: Bearer wai_<your_key>"
```

## Highlights

Highlights are key moments extracted from the analysis — notable quotes, turning points, and critical observations.

### Get Highlights by Analysis

<ParamField path="analysisId" type="string" required>
  The analysis UUID
</ParamField>

```bash
curl -X GET https://api.waterr.ai/v1/highlights/analysis/{analysisId} \
  -H "Authorization: Bearer wai_<your_key>"
```

### Get Highlights by Meeting

<ParamField path="meetingId" type="string" required>
  The meeting UUID
</ParamField>

```bash
curl -X GET https://api.waterr.ai/v1/highlights/meeting/{meetingId} \
  -H "Authorization: Bearer wai_<your_key>"
```

### Generate Highlights

Trigger highlight generation for an existing analysis.

```bash
curl -X POST https://api.waterr.ai/v1/highlights/generate/{analysisId} \
  -H "Authorization: Bearer wai_<your_key>"
```

### Delete Highlights

Remove all highlights for an analysis.

```bash
curl -X DELETE https://api.waterr.ai/v1/highlights/analysis/{analysisId} \
  -H "Authorization: Bearer wai_<your_key>"
```
