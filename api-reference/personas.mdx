## Overview

Personas represent AI characters with specific personality traits, backgrounds, and voice characteristics. They are reusable across multiple scenarios.

## Get All Personas

Retrieve all personas for the authenticated user.

```bash
curl -X GET https://waterr.ai/backend/personas \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

```json
{
  "status": "success",
  "data": [
    {
      "id": "persona-uuid",
      "name": "Sarah Chen",
      "job_title": "Senior Product Manager",
      "demeanor": "friendly",
      "background": "Experienced product manager...",
      "voice_id": 1,
      "avatar": "https://example.com/avatar.jpg",
      "created_at": "2024-01-15T10:30:00Z"
    }
  ]
}
```

## Get Persona by ID

<ParamField path="id" type="integer" required>
  The persona ID
</ParamField>

```bash
curl -X GET https://waterr.ai/backend/personas/{id} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

## Create Persona

Create a new persona with specified characteristics.

<ParamField body="name" type="string" required>
  Name of the persona (max 100 characters)
</ParamField>

<ParamField body="job_title" type="string" required>
  Job title of the persona (max 100 characters)
</ParamField>

<ParamField body="demeanor" type="string">
  Personality style. Options: friendly, analytical, empathetic, curious, enthusiastic, critical, casual, blunt, assertive, neutral
</ParamField>

<ParamField body="background" type="string">
  Detailed background information about the persona
</ParamField>

<ParamField body="gender" type="string">
  `male` or `female` — used for default voice and avatar selection if not specified
</ParamField>

<ParamField body="voice_id" type="string">
  UUID of the voice. Get available voices from the [Voices API](/api-reference/voices). If omitted, a default voice is assigned based on gender.
</ParamField>

<ParamField body="avatar" type="string">
  URL to the persona's avatar image. If omitted, a default avatar is assigned based on gender.
</ParamField>

```bash
curl -X POST https://waterr.ai/backend/personas \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Sarah Chen",
    "job_title": "VP of Engineering",
    "demeanor": "analytical",
    "background": "15 years building distributed systems at scale. Led engineering at two startups through Series B.",
    "gender": "female"
  }'
```

```json
{
  "status": "success",
  "data": {
    "id": "persona-uuid",
    "name": "Sarah Chen",
    "job_title": "Senior Product Manager",
    "demeanor": "friendly",
    "background": "Experienced product manager...",
    "voice_id": 1,
    "created_at": "2024-01-15T10:30:00Z"
  }
}
```

## Update Persona

<ParamField path="id" type="integer" required>
  The persona ID to update
</ParamField>

```bash
curl -X PUT https://waterr.ai/backend/personas/{id} \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Sarah Chen",
    "job_title": "VP of Product",
    "demeanor": "analytical"
  }'
```

## Delete Persona

<ParamField path="id" type="integer" required>
  The persona ID to delete
</ParamField>

```bash
curl -X DELETE https://waterr.ai/backend/personas/{id} \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```
