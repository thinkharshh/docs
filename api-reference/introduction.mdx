The WaterrAI API lets you spin up live AI video meetings, embed them in your product, and pull back transcripts and analysis when the meeting ends.

The fastest way to get started is the [Quick Start](/api-reference/quickstart) — two calls and you have a live meeting room.

## Authentication

Send your API key on every request:

```
Authorization: Bearer wai_<your_key>
```

Get a key at [waterr.ai/settings?tab=api-keys](https://waterr.ai/settings?tab=api-keys). Full details in [Authentication](/api-reference/authentication).

## Base URL

```
https://api.waterr.ai/v1
```

## Response format

Most endpoints return:

```json
{ "status": "success", "data": { ... } }
```

A few (users, meetings) return:

```json
{ "success": true, "data": { ... } }
```

Errors:

```json
{ "status": "error", "message": "Error description" }
```

## Rate limits

Rate limits are applied per API key. Check the response headers:

- `X-RateLimit-Limit` — max requests in window
- `X-RateLimit-Remaining` — requests left in window
- `X-RateLimit-Reset` — when the window resets
